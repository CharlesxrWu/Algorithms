## 排序总结

| 排序    | 平均时间复杂度    |  空间复杂度 |  稳定性 |
| --------| :-----:         | :-----:  | :-----:  |  
| 选择排序 | O(n²)    | O(1)            | 不稳定 |
| 插入排序 | O(n²)    | O(1)            | 稳定 |
| 冒泡排序 | O(n²)    | O(1)             | 稳定 |
| 希尔排序 | O(n1.3)     | O(1)            | 不稳定 |
| 归并排序 | O(nlogn) | 链表O(1) 数组O(n)| 稳定 |
| 快速排序 | O(nlogn) | O(logn)          | 不稳定 |
| 堆排序   | O(nlogn) | O(1) | 不稳定 |
